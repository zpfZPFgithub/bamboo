公开接口

公开接口指的是暴露在bamboo全局变量下的属性方法。整个UI框架只有bamboo变量是暴露在window下的。

接口列表

属性

属性名 类型 介绍 示例
ie6 boolean 判断浏览器版本是否是ie6.

方法

方法名 返回值 介绍 示例
throttle(fn, delay, immediate, debounce) function 创建并返回一个节流函数。当重复调用函数的时候，至少每隔delay毫秒调用一次。在onresize, onscroll, onmousemove等情况下考虑使用 $(window).on("resize", bamboo.throttle(function(){}, 200, false))
debounce(fn, delay, immediate) function 相对于throttle函数的连续delay执行，debounce函数会在最后调用函数delay毫秒之后执行。在onmouseup,不小心点击两次提交按钮等情况下使用 $(window).on("resize", bamboo.throttle(function(){}, 200, false))
drag({}) Drag类的示例 需要拖动某个元素时候，调用bamboo.drag({}) drag({})
resize({}) Resize类的示例 需要缩放某个元素时候，调用bamboo.resize({}) bamboo.resize({})
layer({}) Layer类的示例 弹出窗 bamboo.layer({})
select({}) Select类的示例 自定义滚动条 bamboo.select({})
page({}) Page类的示例 分页 bamboo.page({})
imgcorper({}) Imgcorper类的示例 图片切割程序综合 bamboo.imgcorper({})
